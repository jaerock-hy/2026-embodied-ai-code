# Embodied AI for Autonomous Driving

Course repository for **Embodied AI for Autonomous Driving**.

This project currently contains lecture and practice materials organized by module, including Python exercises and Jupyter notebooks.

## Repository Structure

- `module03/`: Logic gates, object-oriented programming, and related notebook exercises
- `module04/`: Neural network exercises, implementation practice, and tests

## Contents

- Python source files for hands-on exercises
- Jupyter notebooks for interactive learning
- Supporting test and model implementation files

## Getting Started

1. Create and activate a Python virtual environment.
2. Install the dependencies needed for your notebooks and scripts.
3. Run Python files from each module directory as needed.
4. Open notebooks with Jupyter to follow along interactively.
